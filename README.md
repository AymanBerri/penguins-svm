# Penguins SVM Classification

This project uses Support Vector Machines (SVM) to classify penguin species based on body measurements.

## Steps
1. Load and explore the dataset
2. Visualize features
3. Preprocess data (encode labels, standardize)
4. Train Linear SVM
5. Evaluate model
6. Hyperparameter tuning

## Dataset
Seaborn's penguins dataset with 3 species:
- Adelie
- Chinstrap
- Gentoo
